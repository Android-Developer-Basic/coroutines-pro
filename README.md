# Coroutines Pro
Демо-приложение к вебинару Coroutines Pro

## Структура проекта
Проект разбит по шагам на отдельные коммиты - вы можете делать чекаут отдельного
шага и смотреть изменения файлов по каждому шагу

## Содержание

1. Scope, Job (structured concurrency)
2. Cancellation
3. Coroutine Context
4. Обработка ошибок
5. Под капотом
